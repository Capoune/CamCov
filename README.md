簡介
------
使用相機作爲背景。

在路上行走且目光焦於屏幕時更加安全。

由於技術因素，該程序實際上將相機作爲一個半透明的遮罩層。然而表現上和作爲背景區別不大。

類似軟件：transparent screen

可藉由Broadcast/Intent控制，參見[INTEROPERABILITY.md](INTEROPERABILITY.md)。
